# Gaming Skill

Controls the Minecraft gaming section.

## Voice Triggers

- "launch minecraft" / "start minecraft" / "play minecraft"
- "close minecraft" / "stop minecraft"
- "open mods folder" / "open shaders" / "open resource packs"
- "open screenshots" / "open saves"
- "server ping" / "check ping"
- "set up session" / "gaming session"

## Local Only

All functionality is fully local. Server ping uses a raw TCP socket — no external library needed.
